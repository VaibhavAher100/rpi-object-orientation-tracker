# Classifiers

The Haar cascade classifier XML files used in this project
were provided as part of the FAU Erlangen-Nürnberg BiViP
lab course (Laborpraktikum Bild- und Videosignalverarbeitung
auf eingebetteten Plattformen).

They are pre-trained to detect pens in vertical orientation.

Both files are included in this repository:

- `pen_vertical_classifier.xml` — used by detector.py for pen detection
- `haarcascade_frontalface_default.xml` — standard OpenCV face cascade,
