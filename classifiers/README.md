# Classifiers

The Haar cascade classifier XML files used in this project 
were provided as part of the FAU Erlangen-Nürnberg BiViP 
lab course (Laborpraktikum Bild- und Videosignalverarbeitung 
auf eingebetteten Plattformen).

They are pre-trained to detect pens in vertical and 
horizontal orientations.

These files are not included in this repository. To run 
the project, obtain the classifier files from the lab 
course materials and place them here:

- classifiers/pen_vertical.xml
- classifiers/pen_horizontal.xml
